# deployer-recipes

## Installation
* Install composer
```bash
curl -sS https://getcomposer.org/installer | php
```
* Install this package via composer
```bash
php composer.phar require ubiedigital/deployer-recipes
```

## Recipes

### Local
* local:shared

### Symfony
* deploy:create_log_dir
* doctrine:schema:update

### Symfony local
* local:assets:install
